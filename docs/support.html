About the App

Liquid Deadline is a visual deadline manager designed to help you track tasks from start to finish. You can create tasks with titles, categories, notes, start times, and end times, then review them by status, progress, and remaining time.

Key Features

  • Create, edit, and delete deadlines with title, category, description, start time, and end time.
  • View tasks by status: Not Started, In Progress, Completed, and Ended.
  • Filter by category and sort by added time or remaining time.
  • Switch between progress bar view and grid view.
  • Use Home Screen and Lock Screen widgets to see upcoming tasks quickly.
  • Store your tasks and settings locally on your device.

FAQ

  • How do I add a deadline? Open the app and tap the add button to create a new task with its title, category, time range, and notes.
  • How do widgets work? Widgets display nearby tasks based on status, category, and sorting options. Update the widget settings to change what is shown.
  • Is my data synced online? No. The current version stores tasks locally on your device only.
  • Why is a task marked ended? A task moves to Ended when the current time has passed its deadline and it was not manually marked completed before then.

中文

应用简介

Liquid Deadline 是一款专注于截止时间管理的可视化效率应用，帮助你从开始到结束清晰追踪每一项任务。你可以为事项设置标题、分类、备注、开始时间和截止时间，并按状态、进度和剩余时间查看它们。

主要功能

  • 支持创建、编辑和删除事项，包含标题、分类、描述、开始时间和截止时间。
  • 按未开始、进行中、已完成、已结束查看任务状态。
  • 支持按分类筛选，并按添加时间或剩余时间排序。
  • 支持进度条视图和网格视图两种展示方式。
  • 支持主屏幕和锁屏小组件，快速查看临近事项。
  • 任务和设置仅保存在设备本地。

常见问题

  • 如何添加一个 deadline？ 打开应用后，点击新增按钮，填写标题、分类、时间范围和备注即可创建事项。
  • 小组件显示内容可以调整吗？ 可以。你可以在小组件设置中选择状态、分类和排序方式。
  • 数据会上传到云端吗？ 不会。当前版本仅在本地保存任务数据。
  • 为什么任务会显示为“已结束”？ 当当前时间超过截止时间，且任务未被手动标记为完成时，它会进入“已结束”状态。
