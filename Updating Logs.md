# Updating Logs

## Version 1.3
- Time: 2026.03.20
- Compared with Version 1.2:
    - 优化了 iCloud 同步刷新流程，移除了多余的诊断逻辑，减少前后台切换和同步轮询时的干扰
    - 新增提醒编辑与系统提醒调度能力，任务可以配置提醒并下发本地通知
    - 补全了应用和小组件的多语言本地化文案
    - 修复了在设置里打开或关闭自动同步时可能直接闪退的问题，切换后不再需要重新进入应用

## Version 0.2
- Time: 2026.03.09
- Logs:
    - 增加了小组件功能
        - 锁屏小组件：可以显示进行中或将开始的任务名和剩余时间
        - 主屏幕小组件：可以分类显示进行中或将开始的任务，可编辑的有：
            - 状态：进行中/将开始
            - 分类
            - 排序：剩余时间/截止时间
    - 新增了按剩余时间排序的选项

## Version 0.1
- Time: 2026.03.08
- Logs:
    - 可以创建新的任务
        - 标题
        - 分类
        - 描述
        - 起止时间
    - 可以选择不同的排序方式
        - 按截止时间
        - 按添加顺序
    - 可以按照分类筛选显示
    - 可以选择两种不同的显示方式
        - 进度条视图
        - 网格视图
    - 可以设置中文和英文两种语言，默认跟随系统
    - 可以设置三种不同的背景风格
    - 可以编辑分组
    - 可以开关动效
